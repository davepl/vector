# PLTViewer (tekview)

`tekview.py` parses Tektronix 4010/4014 `.plt` vector streams and translates
them into HP 1345A “sendword” commands.

## Plot File Attribution
The `.plt` plot data used for testing and examples is sourced from the Tek4010
project (GPL-3.0+). For example:

```
https://github.com/rricharz/Tek4010/blob/master/pltfiles/16_gplanet.plt
```
